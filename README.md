# Telegram UptoLink Monitor Bot

Bot giám sát link Uptolink và tự động thông báo khi phát hiện mã mới.

## 📌 Tính năng

- ✅ Tự động kiểm tra link Uptolink mỗi 30 phút
- ✅ Phát hiện và trích xuất mã từ link hướng dẫn
- ✅ Thông báo qua Telegram khi có mã mới
- ✅ Thông báo khi link hết mã
- ✅ Hỗ trợ lệnh Telegram: /start, /run, /status, /help
